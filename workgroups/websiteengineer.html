Website Engineer

Role description

Before

  • create a simple landing page to get followers to subscribe for latest updates on PyConTH 2020 before actual website is up.
  • co-ordinate and decide on the how the website should be hosted and using which software for creation.
  • Once decided on the website structure, send website wireframe to design team to prepare assets.
  • Ensure there is a prominent contact email address for most things --contact@pyconthailand.org just to field and route questions to the right person.
  • Ensure that the sponsorship enquiries goes to sponsorship
  • Post conference dates, locations, and Tentative Schedule as early as possible to help cost planning (flights, justification, schedule priorities, etc.)
  • Ensures website contents have been updated on par with social media contents
  • Link Google analytics to website.

During

After

  • Ensures that post event contents will be posted.
  • Ensure that website has been updated with links to videos of talks

Guidelines

  • Is the infrastructure in place for website?
  • Have you called for criticism/suggestions/patches for the website, so it looks professional.
  • Does the website allow easy translation, without one version becoming outdated/out-of-sync.

Keeping track

  • do a close followup on google analytics to keep track on the visit.
  • Ensure that all blog posts and info are consistent with all other media contents.
Also check on step 9 of quick guide for reference.
